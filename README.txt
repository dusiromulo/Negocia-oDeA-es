Para execução do script:

python diretorio_para_run.py comando

Sendo comando:
1: run_idls -> Gera todos os arquivos IDL necessários.
2: run_server -> Executa o servidor, imprimindo as ações iniciais.
3: run_logger -> Executa o logger, imprimindo a ação que foi comprada.
4: run_client -> Executa o cliente, imprimindo os valores das ações naquele momento. Além disso, ele também pede pro usuário digitar o nome de uma ação a ser comprada.
